# 📄 Web 简历系统（Resume Web Project）

## 一、项目概述

本项目为一个基于 Web 的个人简历系统，按照**前端能力提升路径 + 产品迭代思维**进行分阶段设计与实现。

项目从最基础的 HTML 表单与布局出发，逐步演进为：

👉 **可展示 + 可打印 + 可投递的一页 PDF 简历系统**

---

## 二、设计目标与思路

本项目在设计之初即明确：

- 不做“一次性项目”，而是进行**阶段式迭代**
- 每一阶段只解决一类问题
- 每一次升级都有明确技术目标

核心目标：

> ✅ 构建一个“从练习 → 产品”的前端项目

---

## 三、技术演进与版本规划

---

## 🧩 第一阶段（个人简历 1.0）

完成时间：2026.01.24

基于：HTML + CSS（float布局）

📌 设计特点：

- 使用 HTML 表单构建简历填写结构
- 实现基础信息录入（姓名 / 联系方式 / 教育经历等）
- 使用 float 实现页面布局
- 处理浮动塌陷等基础问题

📌 技术目标：

- 理解页面结构
- 掌握传统布局机制

---

## 🧩 第二阶段

完成时间：2026.03.28

📌 更新内容：

- HTML：重构页面结构，新增基本信息展示
- CSS：使用 Flex 替代 float，实现现代布局

📌 核心提升：

👉 从“传统布局” → “现代布局体系”

---

## 🧩 第三阶段

完成时间：2026.03.29

📌 更新内容：

- HTML：
    - 完成简历页面结构与内容填充
    - 优化项目模块层级（项目头 / 标签 / 描述 / 链接）
    - 引入时间轴结构展示经历
    - 新增打印功能入口

- CSS：
    - 构建统一 UI 体系（卡片布局 / Grid / 标签组件 / 技能条）
    - 实现侧边栏渐变风格
    - 新增时间轴组件与 hover 动效
    - 完善打印适配样式

- JS：
    - 实现打印功能（window.print）

📌 核心提升：

👉 从“页面” → “UI系统 + 可打印简历”

---

## 🧩 第四阶段（当前版本）

完成时间：2026.04.10

📌 更新内容：

### HTML：

- 完善项目内容介绍
- 新增项目展示类名（project-links-title / project-links）
- 新增项目截图类名（project-image-title）
- 新增项目截图与项目链接
- 引入 PDF 下载相关第三方库（html2pdf）
- 新增 CSS / JS 版本控制参数（?v=final1）

---

### CSS：

- 重构整体页面排版结构
- 新增项目图片样式
- 新增一页投递模式总开关（one-page-mode）
- 新增侧边栏压缩样式
- 新增主体内容压缩样式
- 新增技能区压缩样式
- 新增内容裁剪规则（隐藏非核心信息）
- 新增打印页面基础样式（@page）
- 重构浏览器打印样式（@media print）

---

### JS：

- 完善打印与 PDF 下载功能
- 实现一页投递模式：
    - 导出前隐藏冗余信息
    - 导出后恢复页面状态

- 实现 PDF 一页排版控制

---

📌 核心提升：

👉 从“可打印” → **“可投递（产品级）”**
